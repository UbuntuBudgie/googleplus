The budgie-remix project has packaged budgie-welcome 0.3.12 and is now available in 16.04 and 16.10 - this new version comes with UI improvements over previous packaged version, ie, v0.3.11.

Following are changes made in this release:

Fix alignment problems associated with INSTALL, DEFAULT SETTINGS and SOFTWARE CENTER. Fix #36
The budgie-welcome now shows version of budgie-desktop installed. This feature can be accessed either using command line arguments --v or -V or using about button in app. Fix #34
The About button is replaced with info icon of material icons. #33
To install budgie-welcome in your budgie-remix system, update your packages.

$ sudo apt update && sudo apt upgrade budgie-welcome

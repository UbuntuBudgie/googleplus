There are elements of Budgie Remix that are indeed very OS-specific. An example of this is the Budgie Welcome screen, it's tailor-made for Budgie Remix, and should obviously remain that way. Previously this stated "is a Solus project", including the all-rights-reserved Solus project branding. I would far rather see Budgie Remix work together with the Budgie crew, than in silos. I would like to see standardisation across all the repos. There is a "gotcha", of course. No official Budgie Desktop git repo should contain any packaging files of any kind. After all, distro agnostic, so no `debian` branches :) It should be noted I'm not just going for "distro-agnostic" here.

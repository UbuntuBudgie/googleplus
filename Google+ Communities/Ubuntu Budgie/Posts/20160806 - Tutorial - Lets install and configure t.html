Tutorial - Lets install and configure this beautiful Conky in budgie-remix :)

• Open your favorite Terminal
• sudo apt install conky-all
• Now you can type conky in your terminal and hit enter, it should appear default conky theme

Following methods are in manual way, also you can get GUI and do this thing in easy way if not just continue below Manual methods :)

• sudo add-apt-repository ppa:teejee2008/ppa
• sudo apt update
• sudo apt install conky-manager
Open conky Manager and have fun !

Manual Methods
-----------------------------------------------------------------------------
Lets add a beautiful Conky theme

• Download the qlocktwo-conky file from this link
http://mowgli-writes.deviantart.com/art/qlocktwo-conky-470067388

• Extract downloaded file and rename it to .conkyrc then open that folder you should see a file called qlocktwo rename it to .conkyrc
• Now move or copy that .conkyrc folder to your Home folder
• Now in Terminal type this
(conky -c ~/.conky/.conkyrc) and hit enter
Now you should see that beautiful conky theme on your desktop.

Lets configure conky with alignments

• Open your favorite Terminal
• type this (gedit ~/.conky/.conkyrc) and hit enter
• Now you should see opened .conkyrc file in gedit. then find rod called alignment usually it's 6th row
• Now make space after the alignment then type your alignment letters in my case it's (alignment tr)

You can configure it any way you like using following alignment letters
• Top left: tl
• Top right: tr
• Top middle: tm
• Bottom left: bl
• Bottom right: br
• Bottom middle: bm
• Middle left: ml
• Middle middle: mm
• Middle right: mr

If you want to get more refined with your theme positioning, you can use the following variables:
• gap_x: Gap, in pixels, from the left or right side of the screen
• gap_y: Gay, in pixels, from the bottom or top of the screen

Lets add a conky to Autostart list

• Open Startup Applications app from Budgie menu or use this command using terminal (gnome-session-properties)
• Now click Add button then, Type name as Conky
• Now as a command type this
(conky -c ~/.conky/.conkyrc) and click Add button.

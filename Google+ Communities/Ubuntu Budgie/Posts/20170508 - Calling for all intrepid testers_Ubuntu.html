Calling for all intrepid testers
Ubuntu Budgie has been working with the creator of the global app-menu (https://github.com/rilian-la-te/vala-panel-appmenu) and we are pleased to have a test PPA with the combined Ubuntu Budgie & Konstantin work in it.

We need you to help out testing please (17.04 only for the moment) - so if you are not afraid of breaking stuff - this is for you.

sudo add-apt-repository ppa:ubuntubudgie-dev/global-menu-test
sudo apt update && sudo apt dist-upgrade
sudo apt install budgie-appmenu-applet

logout and login and add via Raven - Panel "Show menus from windows"

Just for good measure - you can see I have also added budgie-pixel-saver-applet and configured it (via Raven - applet settings) to show just the window controls.

Let us know your thoughts - any real issues please drop by our Gitter community room.

Obviously - no menus will be shown for CSD based apps. GTK+2 apps don't work also (e.g. hexchat) - but GTK+3 apps and others like libreoffice (shown) look ok.

remember to ppa-purge the PPA afterwards to return your desktop back to the supported packageset.

sudo ppa-purge ppa:ubuntubudgie-dev/global-menu-test

cheers

David
